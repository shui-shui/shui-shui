<vector>适用于动态数组的文件头
<unordered_map>是哈希表实现文件头
* std::vector<int> twoSum(std::vector<int>& nums, int target)
我们再解释这句话
std::vector<int> ：动态数组容器，是自动扩容的数组，但容量受系统内存的限制。这个数组里面的数是int类型。
std::vector<char>：这个数组里面的数是char类型的数。
std:string:他就是一串字符
***&的作用（以后再探究）***
* std::unordered_map<int, int> hash_map;
unordered_map表示哈希表，是不保证顺序，属于无序映射。
map属于有序映射
<int, int>第一个int表示数组元素值（key），第二个int是数组下标（value）
* for (int i = 0; i < nums.size(); i++) {
    int complement = target - nums[i];
    // 检查补数是否在哈希表中
    if (hash_map.find(complement) != hash_map.end()) {
        // 找到目标值，返回两个索引
        return {hash_map[complement], i};
    }
    // 将当前元素及其索引加入哈希表
    hash_map[nums[i]] = i;
}
 nums.size()返回nums的长度
